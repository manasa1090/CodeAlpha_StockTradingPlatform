# Stock Trading Platform

## 📖 Project Description
This is a simple Stock Trading Platform project developed as part of my internship. It allows users to view stock data and simulate buying/selling stocks.

## ✨ Features
- View stock list with prices  
- Buy and sell stock simulation  
- Basic portfolio tracking  
- Simple user interface using JavaScript  

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  

## 🚀 How to Run
1. Download or clone the repository  
2. Open `index.html` in a browser  
3. Start using the platform
